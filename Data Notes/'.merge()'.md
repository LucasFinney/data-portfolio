- Most basic way of merging data
- Defaults to an "inner merge" -> A intersect B

Basic Argument Structure:
```
df1.merge(df2,on='id_column', how='inner',suffixes=['_1','_2'])
```
- Use to merge two datasets together along the "on" column
	- If the name of the column doesn't match, use 'left_on =' and 'right_on = ' to specify the separate columns
	- If the column is an index, specify "left_index=True" and/or "right_index=True"
- If there are duplicate column names other than the column you're merging along, can specify the suffix added to each column using the suffixes argument
- Use "how" argument to specify the [[Join Types]]
